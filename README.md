# Lithuania Donation Analysis
An interactive Streamlit application analysing Lithuanian charity donation patterns by municipality, built as part of a university course "Data Mining for Business Intelligence" project at ISM University.

Live app: https://lithuania-donation-analysis-hkfcy5dnmrvqqdizbxmcni.streamlit.app/Geographic_Regional_Patterns 
# Project Overview
This app explores the personal income tax donation dataset from Lithuania's State Data Agency, covering donations made between 2021 and 2023. The analysis focuses on geographic and organisational patterns in charitable giving across Lithuanian municipalities.

# My Contribution
This repository contains my individual contribution to a group project. I was responsible for the geographic and regional analysis, which includes:

- Donation metrics by municipality (total, average, median, standard deviation)
- Top and bottom 10 municipality comparisons across years
- Choropleth map visualisations of donation intensity using log scale
- Donation type share breakdown by municipality
- Most common recipient organisation type by municipality
